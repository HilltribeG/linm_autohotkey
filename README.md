리니지M 매크로
=============

Features
-------------
* 오토핫키를 이용하여 리니지M 에뮬레이터 핸들링
* 비활성 모드에서 사용가능
* 기준 윈도우창 사이즈 : 1280 x 720
* 현재(v1.0.2) 녹스 플레이어, 블루스택3 에서 동작
* 모모 플레이어에서 모든 동작 불등(need adb ...)
* Unicode 32bit compile 후 사용가능
* 관리자 권한으로 실행
